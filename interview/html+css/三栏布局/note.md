#### 优缺点
1. 浮动方案好处：兼容性比较好，缺点可能需要清除浮动
2. 绝对定位：兼容性比较好，缺点因为脱离文档流，可使用性差
3. flex布局：比较完美，兼容性比较差
4. 表格布局：兼容性非常好，单元格高度会同时变化
5. 网格布局：兼容性比较差，代码非常少
